Symmetric Matrix

A Symmetric Matrix is a matrix that is mirrored across the diagonal. Formally, a matrix is symmetric if it equals its transpose: A = AT

Symmetric Matrix

Skew-Symmetric Matrix

A Skew-Symmetric Matrix is a matrix where the lower-triangle is the sign-flipped version of the upper-triangle. Formally: A = -AT

Skew-Symmetric Matrix

Diagonal Matrix

A Diagonal Matrix is a matrix that has all zeros on the off-diagonals, and only the diagonal elements (going from top-left to bottom-right) may contain non-zero elements. Formally: \[ a_{i,j} = \begin{cases} \alpha_n & \text{if } i = j \\ 0 & \text{if } i \neq j \end{cases} \]

Diagonal Matrix

Triangular Matrix

A Triangular Matrix is a matrix that can come in 2 forms:

  • upper-triangular: only the diagonal and the elements above it may contain non-zero elements
  • lower-triangular: only the diagonal and the elements below it may contain non-zero elements
Triangular Matrix

Orthogonal Matrix

An Orthogonal Matrix is a matrix that satisfies the following two criteria:

  1. All of its columns are pairwise orthogonal. It means that the dot product between any two columns is 0.
  2. Each column is unit magnitude. Formally: \( \| Q_i \| = 1 \)
Formally: \[ \langle Q_i, Q_j \rangle = \begin{cases} 1, & \text{if } i = j \\ 0, & \text{if } i \neq j \end{cases} \]

Orthogonal Matrix

Toeplitz Matrix

A Toeplitz Matrix is a matrix in which each descending diagonal contains the same element. Formally: Ai,j = Ai+1,j+1 = ai-j

Toeplitz Matrix

Hankel Matrix

A Hankel Matrix is a square matrix in which each ascending diagonal contains the same element
